# webgl coordinate system

#### introduction

``` javascript
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

// canvas vs webgl

//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//

        // canvas coordinate system
        canvas 的坐标原点默认在在左上角
        canvas 坐标单位是整数
        

                            | y
                            |
                            |
                            |
                            |
                            | (0, 0)
        ------------------- * ------------- + -----------
                            |               |           x
                            |               |
                            |               |
                            + ------------- +
                            |
                            |
                            |


//- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -//


        // webgl coordinate system
        webgl 的坐标原点默认在图形的中心
        webgl 坐标单位是浮点数

                                    | y
                                    |           
                                    |               /
                                    |             / z
                                    |           /
              + ------------------- + ------- / ----------- +
              |                     |       /               |
              |                     |     /                 |
              |                     |   /                   |
              |                     | /  (0.0, 0.0, 0.0)    |
        ----- + ------------------- * --------------------- + -----------
              |                   / |                       |           x
              |                 /   |                       |
              |               /     |                       |
              |             /       |                       |
              + --------- / ------- + --------------------- +
                        /           |
                      /             |
                                    |



//-------------------------------------------------------------------------------------------------------------------//




///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
```
